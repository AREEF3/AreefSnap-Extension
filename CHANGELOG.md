# Changelog

## [1.3.0]
- Renamed extension from AreefSnap to A-Snap
- Added 8 built-in themes: Love, Friends, Ocean, Forest, Candy, Night Owl, Midnight, Simple
- Each theme ships with a matching syntax highlighting style (Atom One Dark, Monokai, GitHub)
- Love theme features animated heart decorations in the background
- Outer and inner background color pickers now reflect the active theme automatically

## [1.0.0]
- Initial release
- Code screenshot with syntax highlighting
- Theme switcher (Dark, Light, Monokai)
- Custom background color
- Save as PNG
