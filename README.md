# AreefSnap 📸

Beautiful code screenshots directly from VS Code.

## Features

- Select any code → Right-click → **AreefSnap: Take Snap**
- Beautiful Mac-style window chrome
- Switch between Dark, Light, and Monokai themes
- Custom background color picker
- Save as high-quality PNG (2x resolution)

## How to Use

1. Select code in the editor
2. Right-click → **AreefSnap: Take Snap**  
   OR open Command Palette (`Ctrl+Shift+P`) → `AreefSnap: Take Snap`
3. Customize theme and background
4. Click **📸 Save PNG**

## Requirements

VS Code `^1.80.0`
