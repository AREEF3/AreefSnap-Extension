# A-Snap 📸

Beautiful code screenshots directly from VS Code.

## Features

- Select any code → Right-click → **A-Snap: Take Snap**
- Beautiful Mac-style window chrome with traffic-light dots
- **8 built-in themes** with matching syntax highlighting
- Custom outer & inner background color pickers
- Save as high-quality PNG (2x resolution)

> The **Love** theme adds glowing heart decorations around the card.

## How to Use

1. Select code in the editor
2. Right-click → **A-Snap: Take Snap**  
   OR open Command Palette (`Ctrl+Shift+P`) → `A-Snap: Take Snap`
3. Pick a theme from the dropdown (or fine-tune with the color pickers)
4. Click **📸 Save PNG**
